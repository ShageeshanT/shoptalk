# ShopTalk MVP Plan

## Goal

Validate whether small businesses want an AI assistant that turns messy customer chats into replies, orders, and follow-ups.

## First niche

Home bakers.

Why:

- Orders are structured
- Timing matters
- Deposits matter
- Custom text and delivery details are common
- Owners already use WhatsApp heavily

## MVP workflow

1. Owner pastes a customer message
2. ShopTalk analyzes it
3. Owner reviews extracted order details
4. Owner edits or copies the suggested reply
5. Owner saves an order or follow-up

## Success signals

- Owner uses it on real customer messages
- Suggested replies save time
- Extracted details are mostly correct
- Follow-ups prevent missed payments or confirmations
- At least 3 businesses say they would pay for it

## Build sequence

1. API skeleton
2. Message analyzer
3. Database models
4. Dashboard UI
5. Follow-up list
6. Order board
7. Pilot feedback
8. Telegram prototype
9. WhatsApp Business API integration
